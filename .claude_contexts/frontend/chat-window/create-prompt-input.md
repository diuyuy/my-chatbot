# PromptInput Component 생성 작업

사용자의 입력을 받고 서버로 사용자의 입력을 전송할 수 있는 `PromptInput` 컴포넌트를 생성해주세요.

**요구 사항**:
1. 상, 하 두 개의 영역으로 구성.
  - 윗 부분: 사용자의 입력에 따라 높이가 자동으로 조절되는 textarea. Max height 설정(적절한 값 직접 입력).
  - 아래 부분: 버튼 목록. 포함할 버튼 목록: Plus 버튼, LLM 모델 선택 버튼, 메시지 전송 버튼.
2. 우선 UI 요소들만 구현. Autosizing되는 Textarea 관련 로직을 제외한 나머지 이벤트 핸들러 및 구체적인 연동 은 구현 불필요 X.
